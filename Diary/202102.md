## CSS layout

웹 페이지에서의 요소들의 배치.

### float

특정 요소가 텍스트나 인라인 요소에 안에 둘러 쌓여 있듯이 배치될 수 있게 해준다.

### flex 

요소들의 정렬을 쉽게 해주는 개념.



## Bootstrap
