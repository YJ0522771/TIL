# [Python] Pandas

* 구조화된 데이터의 처리를 지원.
* 데이터 처리, 통계 분석을 위해 사용.



## 기본

* series : 하나의 column에 해당하는 데이터의 모음.
  * ndarray의 subclass
  * index를 직접 지정 가능
* data frame : table 전체를 포함.
  * series를 모아서 data frame(table)이 됨.
  * indexing
    * loc : index location = 지정된 인덱스 이름 사용.
    * iloc : index position = 지정된 인덱스 말고, 데이터 순서대로 번호를 붙인 인덱스 사용.
